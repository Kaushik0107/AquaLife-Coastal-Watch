# 1. Citizen Mobile Interface

The mobile interface enables citizens to easily report marine pollution and wildlife sightings in coastal areas.

## Login Screen

The login screen allows users to securely access the application.

### Components
- Email input field
- Password input field
- Login button
- Option for new user registration
- Option to Continue with Google account

This screen ensures secure access to the reporting platform.

---

## Home Screen

The home screen acts as the main navigation hub of the application.

### Components
- Quick action cards for main features
- Navigation to reporting and map functionalities
- Basic platform information
  

### Key Actions
- Report Pollution
- Wildlife Sighting Reporting
- View  Map

The home screen helps users quickly access core functionalities.

---

## Report Pollution Screen

This screen allows users to submit pollution reports from coastal areas.

### Components
-  Option to upload Image - directs to upload image Screen
-  Pollution category selection
- Description text field
- Location input or auto-detection
- Submit report button after image upload completion

### Pollution Categories
- Plastic Waste
- Oil Spill
- Garbage
- Dead Marine Life
- others

## Admin Login Screen

The admin login screen allows authorized administrators to securely access the desktop management system of the platform.

### Components
- Admin email input field
- Password input field
- Login button
- Secure authentication validation
- Error message display for incorrect credentials

This screen ensures that only authorized personnel can access administrative tools for monitoring, verifying, and managing coastal pollution reports submitted by citizens.

---























## Dashboard Screen

The dashboard provides a summary of the platform's activity.

### Components
- Total reports submitted
- Number of pollution reports
- Wildlife sightings numbers
- Quick access to Report verification and heatmap tools
- Analytics of Pollution reports over time
- Recent report Highlight

This screen provides an overview of environmental data.

---

## Heatmap Screen

The heatmap screen visualizes pollution density across different coastal regions.

### Components
- Geographic heatmap visualization
- Color-coded pollution intensity
- Identification of major pollution hotspots
- Top pollution area
- Most common waste
- number of reports this week

This helps environmental agencies prioritize affected areas.

---

## Report Verification Screen

The report verification screen allows administrators to review citizen submissions.

### Components
- Table of submitted reports
- Image preview of uploaded evidence
- Location and report type
- Status indicator (Pending / Verified)
- Action buttons for **Approve** and **Reject**

This ensures accuracy and reliability of reported environmental data.

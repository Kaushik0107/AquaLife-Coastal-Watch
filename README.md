# AquaLife Coastal Watch 

## Project Overview

**AquaLife Coastal Watch** is a citizen science platform designed to support **SDG 14 – Life Below Water**. The application enables citizens to report marine pollution and wildlife sightings through a mobile interface while providing environmental agencies with a desktop dashboard to analyze reports, identify pollution hotspots, and verify submitted data.

The system combines **crowdsourced reporting, geo-tagged data, and visual analytics** to help authorities respond quickly to coastal environmental threats.

---

## Features

- **Citizen Reporting System** 
- **Geo-Tagged Image Upload**  
- **Interactive Coastal Map** 
- **Pollution Heatmap Visualization** 
- **Admin Verification Dashboard** 
- **Real-Time Environmental Insights**

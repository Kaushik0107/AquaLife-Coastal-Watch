# AquaLife Coastal Watch

A coastal monitoring platform that helps communities track marine pollution, water quality, and ecosystem health using crowdsourced data and mapping tools.

## Features
- Coastal pollution reporting
- Marine ecosystem monitoring
- Interactive map visualization
- Community alerts
